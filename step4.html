    Step 4: Run the Analysis and View the Results

    1. From the Analysis Settings window, available in the Analysis browser tab, you can choose the location of the output files. Click Run Solver to start the ELS solver. You can check for integrity to help detect any modeling errors that may cause problems in the solution.
    2. After the solution is completed, click Open ELS Results Viewer to view the analysis results and the structure behavior under the defined loading conditions. You can check the data entered in the Preprocessing browser in the Viewer's Input Data browser.
    3. ELS Viewer allows you to create reports, draw charts, and plot contours for the stresses and strains throughout the problem at all simulation steps (frames).
